# Annotation Tools

Enable annotation settings during SDK initialization to allow drawing and highlighting.

Current 3.x releases add keyboard-accessible annotations, snap-to-element behavior, and improved
eraser hit detection. Test keyboard focus, remote-assist ownership changes, and element snapping
with the exact pinned SDK version.
